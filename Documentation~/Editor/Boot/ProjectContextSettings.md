# [`ProjectContextSettings`](/Editor/Boot/ProjectContextSettings.cs)

An extension for the Editor.

Provides a `UnityEditor.SettingsProvider` for the `ProjectSettings` window.

Allows you to edit additional settings via [`AutoGenerateModule`](/Documentation~/Editor/Boot/AutoGenerate/AutoGenerateModule.md) - automatically generates the [`SceneContext`](/Documentation~/Scripts/Boot/SceneContext.md).
