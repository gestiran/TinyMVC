# [`BinderSystem`](/Scripts/Boot/Binding/BinderSystem.cs)

A base class containing a list of [`Binder`](/Documentation~/Scripts/Boot/Binding/Binder.md) objects that are bound as a group.

Simplifies binding multiple [`IDependency`](/Documentation~/Scripts/Dependencies/IDependency.md) to [`SceneContext`](/Documentation~/Scripts/Boot/SceneContext.md).
