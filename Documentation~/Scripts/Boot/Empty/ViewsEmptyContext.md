# [`ViewsEmptyContext`](/Scripts/Boot/Empty/ViewsEmptyContext.cs)

This is a placeholder for creating an empty [`ViewContext`](/Documentation~/Scripts/Boot/Contexts/ViewsContext.md).

Inherits from: [`ViewContext`](/Documentation~/Scripts/Boot/Contexts/ViewsContext.md)

Intended for use in scenarios where the [`SceneContext`](/Documentation~/Scripts/Boot/SceneContext.md) does not contain any [`View`](/Documentation~/Scripts/Views/View.md) components.
This simplifies the context architecture and avoids creating unnecessary classes.
